# Bingo 100 Prototype (Checkpoint: working-tickets)

This snapshot preserves the current working state of the Bingo 100 prototype so you can restore it anytime.

## Includes
- Ticket layout (2×9, 10 numbers, 5 per row)
- Number calling, marking, and auto-call (3s)
- Total Score progress bar with overflow slow-fill + pulse + reset
- Prize UI modal with winners and prize amounts
- CPU players (1–100) with equal ticket counts to the human player
- Responsive UI

## Restore (Git tag)
If you’re using Git, you can also restore to the `working-tickets` tag:
- git checkout working-tickets

## Manual restore
Copy these files back to the project root:
- index.html
- styles.css
- script.js
- README.md

Then open `index.html` in your browser.
